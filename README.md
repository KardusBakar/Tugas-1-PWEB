# Tugas-1-PWEB
TUGAS 1 PWEB Ikhsan Rihan Farli (G1A024083)

Kode HTML tersebut saya buat untuk menampilkan data diri karakter Elaina dalam bentuk halaman web sederhana. Di bagian paling atas ada <!DOCTYPE html> yang berfungsi memberi tahu bahwa file ini menggunakan HTML5. Lalu ada tag <html> yang menjadi pembungkus semua isi halaman.

Di dalam bagian <head> terdapat <title> yang berfungsi untuk memberi judul pada tab browser, yaitu “Data Diri Elaina”. Selain itu, ada juga <style> yang berisi CSS. CSS ini saya gunakan untuk mengatur tampilan supaya halaman tidak terlihat polos. Misalnya, saya mengatur warna latar belakang, jenis huruf, serta membuat sebuah kotak di tengah halaman menggunakan class .kotak. Kotak tersebut diberi lebar, padding, dan border supaya terlihat rapi. Judul dibuat rata tengah, dan gambar juga diatur ukurannya agar tidak terlalu besar dan tetap berada di tengah.

Pada bagian <body>, saya menampilkan isi utama halaman. Di dalam <div class="kotak"> terdapat judul, gambar Elaina, dan beberapa paragraf yang berisi informasi seperti nama, umur, pekerjaan, sifat, dan hobi. Saya menggunakan tag <b> untuk menebalkan kata seperti “Nama” dan “Umur” supaya lebih jelas.

Di bagian bawah, saya menambahkan credit yang berisi link menuju website Wiki Elaina menggunakan tag <a>. Link tersebut bisa diklik dan akan terbuka di tab baru. Secara keseluruhan, kode ini adalah contoh sederhana penggunaan HTML untuk membuat struktur halaman dan CSS untuk mempercantik tampilannya.
